What is HTML?

  • HyperText Markup Language is a notation of tags to create web pages.
  • "Hyper" means that one page can reference other pages.

HTML history

  • First version was designed and implemented by Tim Berners-Lee at CERN
  • Previous mainstream version was 4.01 which appeared in December 1999
  • Development took the abandoned branch with XHTML in 2000-s
  • Current mainstream version 5.0 appeared in October 2014 / 5.1 in October 2015

HTML development

  • Main organization is W3C
  • Alternative organization is WHATWG
  • W3C: each step makes it's way from Nightly Draft -> Working Draft -> Recommendation

Web page structure

  • HTML code define semantics
  • Cascading Style Sheets (CSS) define vizualization
  • Javascript code define behaviour
<html>
  <head>
    <link rel="stylesheet" href="page.css">
    <style>...</style>
  </head>
  <body>
    <!-- ... -->
    <script src="page.min.js"></script>
  </body>
</html>

HTML tag

  • Can either require closure or not
  • Must belong to the list of predefined names (unless Google Polymer is not used)
  • Corresponds to an element in browser's Document Object Model (DOM)
  • Attributes may have no values at all, attribute values may be unquoted if they don't contain spaces or any of " ' ` = < or >
  • Should include appropriate ARIA attributes to make content accessible for to people with disabilities

Build pipeline

  1. Produce CSS files (from SASS, Less, Stylus, etc.)
  2. Apply browser extensions and workarounds to CSS
  3. Minify and merge CSS into single optimized file
  4. Produce JS files (from ES6, from CoffeeScript, from TypeScript, etc.)
  5. Minify and merge JS into single optimized file
  6. Produce media files (raster images, fonts, SVG, video, audio)
  7. Optimize media files (minify, recompress, merge)
  8. Produce HTML files (from Jade, from HAML, from Jinja2, etc.)
  9. Link with resources (CSS, JS, images, etc.)
  10. Minify and merge HTML into single optimized file
  11. Validate HTML
