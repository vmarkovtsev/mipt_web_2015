What is HTML?

  • HyperText Markup Language is a notation of tags to create web pages.
  • "Hyper" means that one page can reference other pages.

HTML history

  • First version was designed and implemented by Tim Berners-Lee at CERN
  • Previous mainstream version was 4.01 which appeared in December 1999
  • Development took the abandoned branch with XHTML in 2000-s
  • Current mainstream version 5.0 appeared in October 2014 / 5.1 in October 2015

HTML development

  • Main organization is W3C
  • Alternative organization is WHATWG
  • W3C: each step makes it's way from Nightly Draft -> Working Draft -> Recommendation
  • WHATWG: living standard
